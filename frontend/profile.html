Privacy Policy Agreement

Our AI-Powered Shopping Assistant captures the keywords, brand names, and price constraints you request in chat or through dashboard filters. This data is indexed locally within your user profile database table in SQLite to allow the recommendation generation engine to query matching deals. We do not transmit clear-text search terms externally unless authorized.

All user passwords are encrypted using state-of-the-art hashing algorithms (Werkzeug security helper module) before storage. Credit card payment actions process transaction reference numbers via internal mock gateways, ensuring actual credit card CVVs and numbers are never persisted in log files or session dumps.

Under GDPR rules, standard customers can withdraw marketing personalization agreements and opt-out of cookie tracking nodes. Opting out will reset recommendation queries back to static general items and will suspend AI matching parameters.
